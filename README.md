# cancer_detection_frontend
This is the **frontend interface** for a Cancer Detection web application. The website allows users to upload medical images (X-rays, CT scans) and get AI-powered predictions for cancer detection.

> **Note:** This repository contains only the frontend (UI/UX) of the application. The backend with the actual ML model is available in the [main repository](https://github.com/pbhavsar5327/cancer-detection).
>
> This repository was created to:
- Showcase the **user interface** and **design** of the cancer detection website
- Demonstrate the **frontend functionality** (image upload, preview, etc.)
- Provide a **live preview** of how the website looks
